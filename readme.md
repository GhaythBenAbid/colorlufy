## Installation

To install and set up the library, run:

```sh
$ npm i colorlufy
```

Or if you prefer using Yarn:

```sh
$ yarn add colorlufy
```

## Usage

```js
const {color} = require("colorlufy");

console.log(color("BgMagenta" , "Hello World"));
```
